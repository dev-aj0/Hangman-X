## Overview

Hangman X is a minimalist web-based version of the classic Hangman game. Players can choose categories, select difficulty levels, and guess letters to reveal hidden words. The game features a clean interface, responsive design, and interactive elements to enhance the user experience.

## Features

- **Category Selection**: Choose from various categories or opt for a random mode.
- **Difficulty Levels**: Select from Easy, Medium, or Hard difficulty settings.
- **Interactive Gameplay**: Use the keyboard or on-screen letters to make guesses.
- **Help Popup**: Access instructions on how to play the game.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Theme Toggle**: Switch between light and dark themes.

## Technologies Used

- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For styling and responsive design.
- **Framer Motion**: For animations and transitions.
- **Radix UI**: For accessible UI components.

## Installation

To get started with Hangman X, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/hangman-x.git
   cd hangman-x
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:8080` to see the app in action.

## Usage

- Start the game by selecting a category or choosing the random mode.
- Pick a difficulty level.
- Guess letters to reveal the hidden word.
- Keep track of your wrong guesses and try to avoid losing the game!
